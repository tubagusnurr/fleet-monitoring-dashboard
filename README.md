# 🚜 Fleet Monitoring Dashboard

## 🎯 Objective

Membangun sistem monitoring fleet berbasis data untuk menganalisis:

* Daftar unit (fleet master)
* Konsumsi bahan bakar
* Utilisasi alat
* Jam operasional

---

## 📊 Key Metrics

* Fuel Consumption
* Utilization Rate
* Operating Hours
* Fleet Availability

---

## 🧩 Data Architecture

1. **listFleet** → Master data unit
2. **fuelFleet** → Konsumsi bahan bakar
3. **utilizationFleet** → Tingkat pemakaian unit
4. **hourFleet** → Jam operasional

---

## ⚙️ Tech Stack

* SQL
* Data Warehouse (Datalake)
* Power BI (Visualization)
